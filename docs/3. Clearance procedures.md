## 3.1 Clearance details

Pilots may be expected to only request for ATC clearance provided they are ready for pushback.

When calling for clearance, pilots may be expected to provide the following: Callsign, destination, ATIS letter, requested flight level, aircraft type, current location / stand, and any special requests / remarks.

If a pilot provides an incorrect ATIS letter, the controller should correct it and ensure the pilot reads it back.

## 3.2 IFR Clearances
Before providing an IFR clearance, the handling controller should ensure that the aircraft's flight plan is valid. The controller should notify and recive a readback from the pilot for all meaningful changes mnade to the flight plan.

IFR Clearances should contain the following information: Cleared airway (if applicable), final altitude / flight level, squawk, and cleared SID.

!!! ifr "Example IFR clearance"
    Sky Surfer 102 (DQA102) cleared to Gan via Z652, FL200, squawk 7433, ATISA1A departure

    “Cleared to Gan via Z652, FL200, squawk 7433, ATISA1A departure, Sky Surfer 102 (DQA102)”

    Sky Surfer 102 (DQA102) readback correct, report ready for pushback and startup with the tow truck connected

!!! warning "Warning"
    Initial climb is NOT provided in an IFR clearance. Instead, it is provided during taxi or takeoff clearance by the applicable controller, as defined in section 5.2.
